<meter value="0.7" min="0" max="1" low="0.2" high="0.8" optimum="0.9">
    70%
</meter>

value="0.7": Represents the current value (70% in this case).
min="0": The minimum value (0%).
max="1": The maximum value (100%).
low="0.2": The value below which the meter is considered low (20%).
high="0.8": The value above which the meter is considered high (80%).
optimum="0.9": The ideal value (90% in this case).

<section>
    <h2>HTML5 Features</h2>
    <p>HTML5 introduces several new features...</p>
</section>

Represents a section of a document, such as a chapter or a group of related content. It helps divide the page into thematic blocks.

    <img src="example.jpg" alt="A beautiful landscape">
    <figcaption>A beautiful landscape in the mountains.</figcaption>
</figure>

<figure>: Represents media content (like an image, illustration, or video) along with its caption.
<figcaption>: Represents a caption for a <figure> element.

<details>
    <summary>More Info</summary>
    <p>This is additional information that can be toggled.</p>
</details>

<details>: Represents a disclosure widget from which the user can obtain additional information or controls.
<summary>: Represents a summary or heading for a <details> element. This is what the user sees initially, and they can click it to reveal the additional information.
